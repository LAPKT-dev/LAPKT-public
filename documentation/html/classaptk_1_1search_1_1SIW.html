std::vector< Action_Idx > &  plan

inlinevirtual

If no partial plan to achieve any goal is found, throw IW(b+1) from same root node

If no state has been pruned by bound, then IW is in a dead-end, return NO-PLAN

If a partial plan extending the achieved goals set is found, IW(1) is thrown from end_state
